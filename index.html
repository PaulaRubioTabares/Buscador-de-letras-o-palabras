Para realizar la búsqueda ingresar la letra o palabra y presionar enter.

El propósito de este proyecto es crear un input que sea capaz de buscar en la página web la palabra que se coloque dentro de este, y a su vez que está palabra se resalte dentro del texto en el que esté, para agregar más contenido a la página web se tiene el siguiente texto: JavaScript es un lenguaje de programación ampliamente utilizado en el desarrollo web. A menudo se le conoce como el lenguaje de la web porque se utiliza principalmente para crear interacciones dinámicas en sitios web. Fue creado originalmente para dar vida a las páginas web, pero permitiendo a los desarrolladores agregar funcionalidades como formularios interactivos, animaciones y actualizaciones en tiempo real.
